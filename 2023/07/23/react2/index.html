react脚手架

脚手架为webpack所搭建，包含了所有需要的配置，并下载了所有相关的依赖

创建项目

  • 全局安装 react 脚手架：npm i -g create-react-app
  • 创建项目： npx create-react-app 项目名
  • 进入文件夹：cd 项目名
  • 启动项目：npm start

部分项目文件分析

  • manifest.json :适配移动端的设置,应用加壳（把网页变成安卓/IOS 软件)
  • robots.txt :规定爬虫爬取的文件规则
  • App.css :App组件的样式
  • App.js :App组件
  • App.test.js :用于给App做测试（使用少）
  • reportWebVitals.js ：页面性能分析文件，需要 web-vitals 库支持
  • setupTests.js ：组件单元测试文件，需要 jest-dom 库支持

index.html

.jsx结尾的都是组件，可与其他js文件区分

axios

安装

npm i axios

配置代理

ps：后端解决跨域问题cors，设置特殊的响应头

方法一：

package.json里加上"proxy": "http://localhost:5000" 重新开启脚手架，请求接口的端口换为本机端口

当请求本机不存在的资源时，就会把请求转发给 5000 端口号服务器

但是只能配单个代理

方法二：

在 src 目录下创建代理配置文件 setupProxy.js
