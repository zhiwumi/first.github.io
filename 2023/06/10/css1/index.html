关于字体文本

字体样式

​ font-family可以指定多种字体。使用多个字体时，将按从左到右的顺序排列，并且以英文逗号（,）隔开。如果我们不定义font-family，浏览器默认字体类型一般是“宋体”。

​ 对于font-family属性，如果字体类型只有一个英文单词，则不需要加上双引号；如果字体类型是多个英文单词或者中文的，则需要加上双引号。

p{font-family:Arial,Verdana,Georgia;}p元素优先使用“Aria字体”来显示，如果你的电脑没有装“Arial字体”，那就接着考虑“Verdana字体”,以此类推。如果Arial、Verdana、Georgia字体都没有安装，那么p元素就会以默认字体（即宋体）来显示。

​ 比较美观的中文字体有微软雅黑、苹方，英文字体有Times New Roman 、Arial和Verdana

关于背景

​ 背景颜色，颜色和透明（transparent）

​ background-image 属性描述了元素的背景图像none和路径url，路径不要加引号(简洁)。实际开发常见于 logo 或者一些装饰性的小图片或者是超大的背景图片, 优点是非常便于控制位置. (精灵图也是一种运用场景)

我们也是使用list-style-type属性来去除有序列表或无序列表的列表项符号的 list-style-type: none;

​ 我们可以使用list-style-image属性来定义列表项图片，也就是使用图片来代替列表项符号。list-style-image: url(图片路径);

​ 背景平铺background-repeat: repeat | no-repeat | repeat-x | repeat-y，横向纵向平铺（默认），不铺，横向，纵向

​ 背景图片位置background-position: x y（40px 30px）或百分数或方位名词（right，bottom center）;

\1. 参数是方位名词

如果指定的两个值都是方位名词，则两个值前后顺序无关，比如 left top 和 top left 效果一致

如果只指定了一个方位名词，另一个值省略，则第二个值默认居中对齐

\2. 参数是精确单位

如果参数值是精确坐标，那么第一个肯定是 x 坐标，第二个一定是 y 坐标

如果只指定一个数值，那该数值一定是 x 坐标，另一个默认垂直居中

\3. 参数是混合单位

如果指定的两个值是精确单位和方位名词混合使用，则第一个值是 x 坐标，第二个值是 y 坐标

​ 背景图像固定background-attachment : scroll | fixed，滚动，固定

​ 背景复合写法background: transparent url(image.jpg) repeat-y fixed top ;

背景色半透明

background: rgba(0, 0, 0, 0.3);

最后一个参数是 alpha 透明度，取值范围在 0~1之间,我们习惯把 0.3 的 0 省略掉，写为 background: rgba(0, 0, 0, .3);

注意：背景半透明是指盒子背景半透明，盒子里面的内容不受影响
